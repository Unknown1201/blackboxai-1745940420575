# CoolDrinks Store

Welcome to **CoolDrinks Store**, your go-to online destination for refreshing cold drinks and convenient bundles. Explore our variety of products that perfectly quench your thirst!

## Project Overview

CoolDrinks Store is a responsive web application that offers a range of popular cold drinks including Sprite, Coca-Cola, Thums Up, and bottled water in various sizes. The platform allows users to easily browse products, add them to a cart, and contact customer support. It features a clean and modern design, built using HTML, CSS, and Tailwind CSS.

## Installation

To run the CoolDrinks Store locally, follow these steps:

1. **Clone this repository:**
   ```bash
   git clone https://github.com/yourusername/cooldrinks-store.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd cooldrinks-store
   ```

3. **Open the `index.html` file in your web browser.**

No additional dependencies are required to view the website as it utilizes CDN links for external libraries.

## Usage

- Open `index.html` in your browser.
- Browse through the hero section and navigate to the products section to view available drinks.
- Click on the "Add to Cart" button for any product you wish to add to your shopping cart.
- Learn more about the CoolDrinks Store in the "About" section and reach out via the "Contact" section.

## Features

- **Responsive Design:** Adaptable layout for various devices.
- **Product Catalog:** Display of multiple cold drinks and bundles for purchase.
- **Interactive Elements:** Functional shopping cart and navigation.
- **Contact Form:** A form for customer inquiries and feedback.

## Dependencies

While this project does not include its own package.json, it utilizes the following external libraries:
- **Tailwind CSS** - for fast and customizable CSS styling.
- **Font Awesome** - for vector icons.
- **Google Fonts** - for custom fonts.

The libraries are included via CDN in the `index.html` file.
